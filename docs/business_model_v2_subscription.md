# 3rd Place ビジネスモデル v2：サブスク型（検討案）

*作成日: 2026-04-08*

---

## 背景：なぜ現モデルを見直すか

### 現モデルの課題
- パッケージ A$150-200 は渡航前の段階では心理的ハードルが高すぎる
- 「情報」と「サービス」が一括りになっており、何にいくら払ってるか不明
- 渡航サポート（Zoom、レジュメ添削）はエージェントが無料で提供しており差別化が弱い

### 留学エージェント業界の構造
- 無料エージェント（StudyIn, スマ留等）は語学学校からのコミッション（授業料の15-30%）で運営
- 有料エージェント（夢カナ留学）はサポート料80-90万円 + コミッション
- いずれも「語学学校に送る」ビジネス → **語学学校に行かないワーホリ勢は取りこぼされている**
- 3rd Placeはエージェントの競合ではなく、**エージェントがカバーしない領域（仕事・住居の現場情報）を埋める存在**

### 3rd Placeの唯一の競合優位
**引き継ぎノート（データベース）** — エージェントにもGaijinPotにもFacebookにもない、メンバー限定の構造化された現場情報。

## 新モデル：サブスク + トランザクション課金

### 情報の3層モデル

```
┌──────────────────────────────────────────────────┐
│  Layer 1: 公開コンテンツ（ブログ）                     │
│  匿名化・一般化した体験談。SEO集客用                     │
│  → Google検索で誰でも見れる                           │
├──────────────────────────────────────────────────┤
│  Layer 2: Hub概要（LINE参加で閲覧可）                  │
│  業界別ガイド概要、Zoomイベント見学、カレー会情報           │
│  → 無料だがコミュニティに入る必要あり                     │
├──────────────────────────────────────────────────┤
│  Layer 3: 引き継ぎノート + ライブZoom（月額メンバー限定）  │
│  具体的な店の雰囲気、オーナーの人柄、初週のコツ             │
│  先輩ゲストスピーカーとの少人数Zoom                      │
│  → 月額課金                                         │
└──────────────────────────────────────────────────┘
```

### キャッシュポイント

#### CP1: 月額サブスク（A$10-15/月）— 情報 + コミュニティアクセス

| 内容 | 詳細 |
|------|------|
| 引き継ぎノート全文閲覧 | 仕事・住居の現場情報（読み放題） |
| 業界ガイド詳細版 | カフェ・ジャパレス・クリーニング・マッサージ等 |
| メンバー限定Zoom（月2-4回） | 業界別・テーマ別の少人数ゲストスピーカー会 |
| 新着ノート通知 | 新しい引き継ぎノート・情報更新のプッシュ通知 |
| レジュメテンプレート | メルボルン向けにカスタマイズ済み |
| パートナー店舗紹介状 | 「3rd Placeメンバーです」の証明 |

**性質**: ストック型（解約まで積み上がる）

**無料Zoomとの差別化**:
- 無料：月1回のオープンZoom（一般テーマ、誰でも参加可、集客用）
- 有料：月2-4回の少人数Zoom（業界別、質問OK、メンバー限定）

**サブスクの継続理由**:
- ノート（ストック）だけだと「読んだら解約」のリスク
- ライブZoom（フロー）があることで「来月も聞きたい」が生まれる
- Netflix/Spotifyと同じ構造：過去ライブラリ + 新作コンテンツ

#### CP2: 住居バトンパス仲介（A$50-100/件）— トランザクション課金

```
帰国する先輩 → 自分の部屋を後輩に引き継ぎたい
到着した後輩 → 今すぐ住む場所が欲しい（ホステル1泊A$40-60）
3rd Placeが接続 → 成約時にA$50-100
```

- 到着直後は緊急性が最も高く、支払い意欲がピーク
- 自力で探すと1-2週間（ホステル代A$400-800）→ バトンパスの方が安い
- 家具付き・日本人前住者・大家の性格も把握済み → リスクが極めて低い
- エージェントは住居紹介をやらない（語学学校がキャッシュポイントだから）

#### CP3: パートナー店舗紹介（成果報酬 A$20-30/件）— コネクション課金

- メンバーがパートナー店舗に紹介状を持って行く → トライアル獲得
- 採用決定時にA$20-30（1-2時間分の時給。1日早く仕事が見つかれば元が取れる）
- 「マッチング確約」ではなく「接続の機会」（既存原則の遵守）
- 成果報酬なので「紹介してもらったけど落ちた」リスクがない

#### CP4: コミュニティ告知枠（A$50-200/回）— BtoB

- パートナー店舗や現地ビジネスの告知をHubに掲載
- カレー会でのブース出展
- Phase 3以降で本格化

## キャッシュバック設計（循環エンジン）

| アクション | キャッシュバック | 備考 |
|-----------|-------------|------|
| 到着後アンケート | A$10 | 初回の接点 |
| 近況報告（3ヶ月ごと） | A$5/回 | 情報の鮮度維持 |
| ゲストスピーカーZoom出演 | A$5/回 | スピーカー確保のインセンティブ |
| 引き継ぎノート作成 | A$30 | 最重要。循環の核 |
| 住居バトンパス提供 | A$50 | 退去者にも報酬 |
| 後任紹介 | A$50 | リファラル |

月額3ヶ月分 = A$36 に対してノート作成キャッシュバック = A$30。
→ 情報提供者は実質 A$6 で3ヶ月間ノート読み放題だった計算。

## ファネル全体フロー

```
① Google「メルボルン ワーホリ カフェ 仕事」
     ↓
② ブログ記事（無料）— 時給相場、エリア情報、体験談のエッセンス
     ↓
③ LINE公式アカウント追加（無料）— 通知チャネル
     ↓
④ Hub閲覧 + 月1オープンZoom見学（無料）— 信頼構築
   「先輩がリアルに話してる。このコミュニティは本物だ」
     ↓
⑤ 月額メンバー（A$10-15/月）— 最初の課金ハードルが低い
   「ノート全文 + 少人数Zoom + 紹介状」
     ↓
⑥ 渡航・到着
   住居バトンパス（A$50-100）/ 店舗紹介（A$20-30）
     ↓
⑦ カレー会参加（永続無料）
     ↓
⑧ 滞在中：ミニレビュー / ゲストスピーカー出演
   → キャッシュバック
     ↓
⑨ 帰国前：引き継ぎノート作成 / 住居バトンパス提供
   → キャッシュバック
     ↓
⑩ ノート → ブログ記事の素材（①に戻る。循環完成）
```

## コンテンツ変換パイプライン

引き継ぎノート1件から3層すべてにコンテンツを供給。提供者の負担は増えない。

```
引き継ぎノート（1件）
  ├── → Layer 3: そのまま格納（月額メンバー限定）
  ├── → Layer 2: 集計データに反映（平均時給の更新等）
  └── → Layer 1: ブログ記事の素材としてストック
        （5〜10件溜まったらテーマ別に1記事にまとめる）
```

## プラットフォーム構成

```
LINE公式アカウント ── 入口・通知チャネル（無料、全員）
  │                  新着ノート通知、カレー会告知、Zoom案内
  │
  ├→ Hub（Webアプリ）── 情報・課金の中心
  │                     引き継ぎノート、業界ガイド、検索、地図
  │                     月額メンバーのアクセス管理
  │
  ├→ ブログ（Webサイト）── SEO集客
  │                      匿名化した体験談、ガイド記事
  │
  └→ カレー会 / Zoom ── コミュニティ（無料、全員 + メンバー限定）
```

現在のLINEグループは LINE公式アカウント に移行。
- 運営→メンバーへの一方向配信（通知・お知らせ）
- メンバー→運営への1on1チャット（質問・相談）
- マサさんの長文が全員に流れる「教祖感」問題が解消される

## 収益シミュレーション

### Phase 2（6ヶ月後）

```
月額メンバー:      50人 × A$12   = A$600/月
住居バトンパス:    月2件 × A$75  = A$150/月
店舗紹介:         月3件 × A$25  = A$75/月

合計: A$825/月
```

```
月額メンバー:      200人 × A$12  = A$2,400/月（ストック累積）
住居バトンパス:    月5件 × A$75  = A$375/月

告知枠(BtoB):     月2件 × A$100 = A$200/月
───────────────────────────────────────
合計: A$3,225/月
```

## エージェントとの関係：競合ではなく補完

```
エージェントの領域:
  語学学校選び → ビザ申請 → 航空券 → 保険 → 出発

3rd Placeの領域:
  仕事の探し方 → 住居の探し方 → 現地の人間関係 → 生活の立ち上げ

時間軸:
  ──渡航前──┬──到着──┬──生活開始──┬──帰国前──
  エージェント│        │            │
  ═══════════╡        │            │
             │3rd Place│            │
             │════════════════════════
```

無料エージェント（StudyIn等）との提携も可能。
「語学学校卒業後の仕事・住居探しは3rd Placeへ」と案内してもらえれば双方にメリット。

## 既存原則との整合性チェック

| 原則 | 整合性 |
|------|--------|
| エージェントに依存しない | ◯ エージェントと補完関係。依存ではない |
| コミュニティ内にお金を見せない | ◯ カレー会は無料。課金はHub内で完結 |
| 属人性を排除 | ◯ Zoomゲストは毎回変わる。特定の人に依存しない |
| カレー会は永続無料 | ◯ 変更なし |
| 情報提供者はメンバー限定 | ◯ 変更なし |
| 口コミサイト化しない | ◯ 構造化されたフォーム投稿。自由記述ではない |

## 未解決の論点

- 月額課金の決済インフラ（Stripe等）の実装
- ゲストスピーカーの持続的な確保方法
- 住居バトンパスの法的整理（転貸契約のルール等）
- 月額の具体的な価格設定（A$10 vs A$12 vs A$15）
- コールドスタート：最初の引き継ぎノートをどう集めるか（運営10人が自分で書く？）
- マサさん（教授）の合意
