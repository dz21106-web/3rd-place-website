# 3rd Place — 事業概要書

作成日：2026年4月3日

---

## 1. 3rd Placeとは

メルボルンに渡航する日本人（ワーキングホリデー・留学生）に向けて、現地で実際に働いた・住んだ先輩の「生の経験情報」を提供するコミュニティ型プラットフォームです。

**一言で言うと：**
> 「先輩の経験を、次の人の武器にする」

留学エージェントのように企業側から手数料を取るモデルではなく、渡航者本人に価値を提供し対価をいただくBtoCモデルです。

---

## 2. 解決する課題

メルボルンに来る日本人が直面する問題：

| 課題 | 現状 |
|------|------|
| 仕事探し | 求人サイトには載らない「どんな職場か」がわからない。レジュメを配り歩くしかない |
| 住居探し | Facebookや掲示板で探すが、大家の性格・ハウスメイトの雰囲気は住むまでわからない |
| 情報の信頼性 | ネット上の情報は古い・一般的すぎる。実際に経験した人の声が一番信頼できるが、つながる手段がない |
| 到着直後の不安 | 知り合いがいない・土地勘がない・英語に不安がある中で、すべてを自力でやるしかない |

**3rd Placeの解決策：**
先輩が残した引き継ぎノート（職場の実態・住居のリアルな情報）と、業界別の集合知を武器として渡し、「自分で取りに行ける人」に育てる。

## 3. 提供サービス

### メインサービス：メルボルン着パッケージ（A$150〜200/人）

| 提供内容 | 詳細 |

## 4. ビジネスモデル

### なぜBtoCか

メルボルンの飲食業界は求職者過多で、店舗側はレジュメが勝手に届くため採用に費用をかける動機がありません。痛みを持つのは求職者側（渡航者）であり、彼らに直接価値を提供して対価をいただくモデルです。

### 収益構造

```
メイン収益：メルボルン着パッケージ（A$150〜200/人）
サブ収益①：コミュニティ告知枠（A$50〜200/回）
サブ収益②：長期会員制（A$20/月）
将来収益：住居バトンパス仲介
```

### キャッシュバック設計（利用者の実質コスト削減＋情報循環の仕組み）

| タイミング | アクション | キャッシュバック |
|-----------|-----------|--------------|
| 到着1〜2週間後 | 到着後アンケート回答 | A$10 |
| 3ヶ月ごと | 近況報告・情報更新 | A$5/回 |
| 帰国前1〜2ヶ月 | 引き継ぎノート作成 | A$30 |
| 任意 | 後任メンバー紹介 | A$50 |

12ヶ月フル滞在の実質コスト：**A$55〜105**

このキャッシュバック設計により、利用者が自然と情報の提供者にもなる循環モデルが成立します。

## 5. 競合優位性

### 引き継ぎノート＋業界別集合知

求人サイトにも不動産サイトにも絶対に載らない「経験者しか知らない情報」が蓄積されます。

- 職場の引き継ぎノート：オーナーの性格・採用で見られるポイント・最初の1週間でやるべきこと・NGなこと
- 住居の引き継ぎノート：大家の人柄・ハウスメイト情報・入居初日の注意点
- 業界別ガイド：複数メンバーの経験を集約した業界ごとの集合知（時給相場・採用パターン・暗黙ルール・危険信号）

**データが蓄積されるほど価値が高まり、後発が追いつけない参入障壁になります。**

### 既存の選択肢との違い

| 選択肢 | 3rd Placeとの違い |
|--------|------------------|
| 留学エージェント | 企業側から手数料を取るため、利用者の利益より提携先の利益を優先しがち。3rd Placeは利用者から直接対価をもらうので利害が一致 |
| Facebookグループ | 情報が流れて蓄積されない。誰でも書けるので信頼性が低い。3rd Placeはメンバー限定の構造化された情報 |
| ブログ・YouTube | 一般的な情報は得られるが、特定の職場・住居のリアルな実態はわからない |
| 口コミサイト | 店舗側のメリットが消える設計（3rd Placeは口コミサイト化を避け、ライトな情報共有に留める） |

## 7. サービスの位置づけ（重要）

3rd Placeは**求人の斡旋・仲介を行いません**。

| やること | やらないこと |
|---------|------------|
| 業界の実態情報を提供する | 特定の求人に応募者を送り込む |
| 先輩の経験を構造化して共有する | 採用やトライアルを確約する |
| 先輩メンバーとの接続機会を提供する（運営がZoomをセッティング） | 利用者と店舗の間に入って交渉する |
| レジュメ添削・推薦状を発行する | 雇用契約に関与する |

提供するのは「情報・武器・繋がり」であり、実際の就職・入居は利用者本人が行います。

## 10. 絶対原則

1. **エージェントに依存しない**（OEM・下請けにならない）
2. **コミュニティ内にお金を見せない**（人間関係がドライになるのを防ぐ）
3. **属人性を排除する**（仕組みとして自走できる設計）
4. **カレー会は永続無料**（まごころ精神。過去に有料化して失敗した経験あり）
5. **創設者（マサ氏）の合意なく進めない**
6. **情報提供者はメンバー限定**（外部開放すると信頼性がゼロになる）

## 11. 現在の課題と対策

### 最緊急：ブランドイメージの改善

2026年3月の現地ヒアリング（5名＋運営会議）で、「中身は素晴らしいが見せ方が間違っている」という指摘が全員一致で出ています。具体的には：

- LINEでの長文アナウンスが「宗教っぽい」印象を与えている
- 完全無料のイベントへの不信感（「ただより怖いものはない」）
- ウェブサイトがなく、外から活動内容が見えない
- 女性が一人では参加しにくい雰囲気

**対策：** LINE発信のトーン調整、公共の場でのイベント開催、ウェブサイト/LP作成、一部イベントへの少額参加費導入

### データベースの情報収集

引き継ぎノートの価値は全員が認めているが、「情報をどう集めるか」が最大の未解決課題。

**対策：** Googleフォームでの構造化された収集、キャッシュバックによるインセンティブ設計、帰国前メンバーへのヒアリング

## 12. アプリ：3rd place hub

Webアプリとして開発中。3つの機能で構成：

| 機能 | 内容 | 対象 |
|------|------|------|
| **知る** | 業界別ガイド（集合知）・週1 Zoomイベント一覧 | 渡航前の準備段階 |
| **探す** | 職場・住居の個別プロフィール（複数メンバーの集合データ） | 到着前〜到着直後 |
| **引き継ぐ** | 引き継ぎノートの入力フロー（Googleフォーム連携） | 帰国前 |

業界ガイドは無料公開し、引き継ぎノートの詳細はパッケージ購入者限定で閲覧可能にすることで、課金導線として機能させます。
