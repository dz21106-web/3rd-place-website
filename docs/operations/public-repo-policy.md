# 公開リポジトリ運用ポリシー

最終更新: 2026-04-17

## 前提
- このリポジトリは公開前提で運用する。
- コード・デザイン資産・公開可能な事業資料のみを置く。

## コミットしないもの
- 個人情報を含む資料
- 社内向けメモ、議事録、日次作業ログ
- 未公開の事業資料、営業資料、提案書
- APIキー、トークン、DB ID、秘密情報
- 顧客情報、連絡先、契約情報

## 配置ルール
- 公開可能な運用ルールは `docs/operations/` に置く。
- 非公開資料は Git 管理しない。必要なら `docs/private/` をローカル専用で使う。
- 従業員向けメモや個人情報を含むファイルはリポジトリに置かない。

## Git ルール
- `.env`, `.env.local` などの実値ファイルはコミットしない。
- `docs/employees/` と `docs/operations/*-work-log.md` は ignore 対象として扱う。
- 公開して問題ないか迷うファイルはコミット前に止める。

## 公開前チェック
1. 個人名、メール、電話番号、住所が含まれていないか確認する
2. APIキーや URL パラメータ付き秘密情報が入っていないか確認する
3. 社内向け表現や未公開判断が残っていないか確認する
4. 公開不要の PDF / PPTX / 議事録が含まれていないか確認する
