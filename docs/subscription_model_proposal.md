# 3rd Place ビジネスモデル再設計案：サブスクリプション型

> 運営メンバー共有用ドキュメント（2026-04-08作成）
> ステータス：検討中の案。決定事項ではありません。

---

## なぜ現モデルを見直すのか

### 現モデルの課題

現在の「着パッケージ A$150〜200」には構造的な問題がある。

**1. 価格ジャンプが大きすぎる**

```
無料（ブログ/LINE/Hub閲覧/カレー会） → いきなり A$150-200
```

ファネルの途中まで無料なのに、突然A$150を求める。ワーホリ層（18-30歳、予算タイト）にとって心理的ハードルが高く、「まあ自分で何とかなるか」と離脱される。

**2. 情報とサービスが混ざっている**

パッケージの中身を分解すると：

| 中身 | 性質 | 原価 |
|------|------|------|
| 引き継ぎノート閲覧 | 情報（複製コストゼロ） | ほぼ0 |
| 業界ガイド詳細 | 情報 | ほぼ0 |
| Zoomブリーフィング | サービス（人の時間） | 高い |
| レジュメ添削 | サービス | 高い |
| 推薦状 | サービス | 中 |
| 到着後フォロー | サービス | 高い |

情報は安く広く売るべき。サービスは高く少なく売るべき。一括りにすると値付けがおかしくなる。

**3. 渡航サポートはエージェントと被る**

| サポート項目 | 留学エージェント | 3rd Place独自？ |
|------------|-------------|--------------|
| Zoomカウンセリング | 無料でやってる | × |
| レジュメ添削 | 夢カナ等がやってる | × |
| 推薦状 | やらない | ◯（パートナー店舗ネットワーク） |
| 到着後フォロー | 現地エージェントがやってる | △ |

StudyIn等の無料エージェントが「無料で」やっていることを、有料で売ろうとしている。

## 留学エージェント業界の構造（リサーチ結果）

### 主要プレイヤー

| エージェント | 利用者への料金 | 収益源 |
|------------|-------------|--------|
| StudyIn | 無料 | 語学学校からのコミッション（授業料の15-30%） |
| 夢カナ留学 | 80〜90万円 | サポート料 + 語学学校コミッション |
| スマ留 | 無料〜6万円 | 語学学校コミッション |
| タビケン留学 | 無料 | 語学学校コミッション |

### エージェントの構造的弱点

無料エージェントは語学学校にユーザーを送ることで収益を得ている。つまり：
- **語学学校に行かないワーホリ勢には価値を提供できない**
- コミッションの高い学校を推す構造的インセンティブがある
- 語学学校卒業後の「仕事・住居探し」はサポート範囲外

### 3rd Placeが埋める空白

```
エージェントの領域:
  語学学校選び → ビザ申請 → 航空券 → 保険 → 出発

3rd Placeの領域:
  仕事の探し方 → 住居の探し方 → 現地の人間関係 → 生活の立ち上げ
```

エージェントと競合するのではなく、**エージェントがカバーしない領域を埋める存在**。

## 提案：サブスクリプション型モデル

### キャッシュポイントは「データベースへのアクセス権」一本

3rd Placeがエージェントに絶対勝てる唯一の資産は**引き継ぎノート（データベース）**。

- 語学学校コミッションで食ってるエージェントには、「卒業後の仕事・住居のリアル情報」を集めるインセンティブがない
- Facebookや口コミには構造化されたデータがない
- GaijinPotには「メルボルンの日本人ワーホリ向け」の深さがない

### 2層の料金構造

```
┌──────────────────────────────────────────┐
│  無料（集客 + 信頼構築）                      │
│                                          │
│  ・ブログ記事（SEO集客）                     │
│  ・LINE公式アカウント（通知・入口）             │
│  ・Hub「知る」タブ概要                       │
│  ・カレー会参加（永続無料）                    │
│  ・月1回オープンZoom（一般テーマ）              │
├──────────────────────────────────────────┤
│  月額 A$10-15（メンバーシップ）                │
│                                          │
│  ・引き継ぎノート全文閲覧                      │
│  ・業界ガイド詳細版                          │
│  ・メンバー限定Zoom（月2-4回、業界別ゲスト）     │
│  ・新着ノート通知                            │
│  ・メンバー限定チャット                       │
│  ・レジュメテンプレート                       │
│  ・パートナー店舗紹介状                       │
└──────────────────────────────────────────┘
```

### 段階的なキャッシュポイント（追加収益源）

月額サブスクに加え、ユーザーの旅の中で自然に発生する課金ポイント：

**住居バトンパス仲介（A$50-100/件）**
- 帰国する先輩の部屋を到着した後輩に引き継ぐ
- 到着直後は緊急性が最も高い（ホステル泊A$40-60/泊）
- 引き継ぎなら家具付き・日本人前住者・大家の性格も分かってる
- エージェントも不動産サイトもカバーしない領域

**パートナー店舗紹介（成果報酬 A$20-30/件）**
- メンバーが紹介状を持って店に行く → 採用が決まったら課金
- 成果報酬なので「紹介してもらったけど落ちた」リスクなし
- 「マッチング確約」ではなく「接続の機会」（既存原則の遵守）

**コミュニティ告知枠（A$50-200/回）**
- パートナー店舗・現地ビジネスからのBtoB収益
- Phase 3以降で本格化

## メンバー限定Zoom（ゲストスピーカー制）

月額メンバーシップの**目玉コンテンツ**。

### なぜこれが強いか

| | エージェントのZoom | 3rd PlaceのZoom |
|---|---|---|
| 話す人 | 営業スタッフ | 今まさにメルボルンで働いてる先輩 |
| 内容 | 営業トーク | 昨日あった出来事、リアルな日常 |
| 再現性 | どのエージェントもやれる | メンバーネットワークがないと不可能 |

### 設計

```
無料: 月1回オープンZoom（一般テーマ、集客用）
      「メルボルンのワーホリ生活ってどんな感じ？」

月額限定: 月2-4回の少人数Zoom（5-10人）
      「カフェで働いてる先輩に直接質問できる会」
      「シェアハウス探しで失敗しないコツ — 実体験トーク」
```

### メリット
- 引き継ぎノートの「ライブ版」。テキストでは伝わらないニュアンスが伝わる
- 実在する人が顔を出して話す → 「宗教っぽさ」が消える
- ゲスト側のコストほぼゼロ（自分の経験を話すだけ）
- 録画→匿名編集→ブログやSNSの素材に再利用可能

### 月額サブスクとの相性
- ノートだけだと「一回読んだら解約」のリスクがある
- ライブZoomがあると「来月も聞きたい」が生まれ、解約防止になる
- 月額の価値 = ノート読み放題（ストック）+ ライブZoom（フロー）

## ファネル全体のフロー

```
① Google検索「メルボルン ワーホリ カフェ 仕事」
     ↓
② ブログ記事を読む（無料）
     ↓
③ LINE公式アカウント追加（無料）
     ↓
④ Hub閲覧 + オープンZoom参加（無料）
   「信頼できそう。もっと詳しく知りたい」
     ↓
⑤ 月額メンバー登録（A$10-15/月）
   ノート全文 + 限定Zoom + 紹介状
     ↓
⑥ 渡航・到着
   住居バトンパス（A$50-100）/ 店舗紹介（A$20-30）
     ↓
⑦ カレー会参加（永続無料）
     ↓
⑧ 滞在中：限定Zoomにゲスト出演（A$5報酬）
     ↓
⑨ 帰国前：引き継ぎノート作成（A$30キャッシュバック）
   住居バトンパス提供側（A$50報酬）
     ↓
⑩ ノート → ブログ記事の素材に（①に戻る、循環完成）
```

## 収益シミュレーション

### Phase 2（6ヶ月後）

```
月額メンバー:     50人 × A$12  =  A$600/月
住居バトンパス:   月2件 × A$75  =  A$150/月
店舗紹介:        月3件 × A$25  =   A$75/月
────────────────────────────────────────
合計: A$825/月
```

### Phase 3（12ヶ月後）

```
月額メンバー:     200人 × A$12 = A$2,400/月（ストック型で積み上がる）
住居バトンパス:   月5件 × A$75  =  A$375/月
店舗紹介:        月10件 × A$25 =  A$250/月
告知枠(BtoB):    月2件 × A$100 =  A$200/月
────────────────────────────────────────
合計: A$3,225/月
```

→ **Phase 3 KPI（A$2,000-3,000/月）を達成可能**

### 旧モデルとの比較

```
旧: パッケージ A$175 × 月10人 = A$1,750（ただし転換率が低い）
新: サブスク中心で A$3,225（ストック型で安定、転換率が高い）
```

## キャッシュバック設計の再整理

| タイミング | 行動 | 報酬 |
|-----------|------|------|
| 滞在中（随時） | 限定Zoomにゲスト出演 | A$5/回 |
| 滞在中（3ヶ月ごと） | 近況ミニレビュー | A$5/回 |
| 帰国前 | 引き継ぎノート作成 | A$30 |
| 帰国前 | 住居バトンパス成立（提供側） | A$50 |
| 随時 | 後任メンバー紹介 | A$50 |

月額3ヶ月分 = A$36。引き継ぎノート作成だけでA$30返ってくるので、**実質A$6で3ヶ月間ノート読み放題だった計算**。情報提供者には「元が取れた」体験を提供できる。

## プラットフォーム構成

現在のLINEグループに全機能を押し込んでいる状態から、役割を分離する。

```
LINE公式アカウント = 通知・入口（無料、全員）
  ・新着ノート通知、カレー会告知、Zoom案内
  ・1on1チャットで質問対応
  ・マサさんの長文が全員に流れる問題を解消

Hub（Webアプリ）= 情報・課金（月額メンバー）
  ・引き継ぎノート、業界ガイド、検索、地図
  ・ここが課金ポイント

ブログ（Webサイト）= SEO集客（無料、全員）
  ・引き継ぎノートを匿名化・一般化した記事
  ・Googleから新規ユーザーを引き込む

カレー会 / Zoom = コミュニティ（無料 + メンバー限定）
  ・オープンZoom月1（無料、集客用）
  ・限定Zoom月2-4（メンバー限定、業界別ゲスト）
```

## コンテンツ変換パイプライン

引き継ぎノート1件から3つのアウトプットを生成。提供者の負担は増えない。

```
引き継ぎノート（1件）
  │
  ├── → Hub にそのまま格納（月額メンバー限定）
  │
  ├── → Hub の集計データに反映（平均時給の更新等）
  │
  └── → ブログ記事の素材としてストック
        （5〜10件溜まったらテーマ別に1記事にまとめる）
```

## エージェントとの関係

競合ではなく**補完関係**を目指す。

```
時間軸:
  ──渡航前──┬──到着──┬──生活開始──┬──帰国前──
  エージェント│        │            │
  ═══════════╡        │            │
             │3rd Place│            │
             │════════════════════════
```

将来的にStudyIn等の無料エージェントとの提携も可能。「語学学校卒業後の仕事・住居探しは3rd Placeへ」と案内してもらえれば、エージェント側も顧客満足度が上がる。

## 変わること / 変わらないこと

### 変わること
- パッケージ一括払い → 月額サブスク + トランザクション課金
- LINEグループ → LINE公式アカウント（通知特化）
- 渡航サポート（Zoom/レジュメ）を有料で売る → 月額特典に含める or 廃止
- 引き継ぎノートは「パッケージ購入者限定」→「月額メンバー限定」

### 変わらないこと
- カレー会は永続無料
- 情報提供者はメンバー限定
- 口コミサイト化しない
- コミュニティ内にお金を見せない
- 「マッチング確約」「トライアル確約」は言わない
- 教授（マサ氏）の合意なく進めない
