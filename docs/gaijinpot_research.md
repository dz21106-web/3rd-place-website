# GaijinPot リサーチ — 3rd Place への示唆

## GaijinPot 概要

- **正式名称**: GaijinPot（ガイジンポット）
- **運営**: GPlusMedia Inc.（2022年〜学研ホールディングス傘下）
- **創業**: 1999年（外国人英語教師2人が立ち上げ）
- **規模**: 月間75万ユーザー、登録履歴書47万件、年商約$6M
- **対象**: 日本に住む外国人（主に英語圏）
- **サービス**: Jobs / Apartments / Study / Travel / Events / Blog

---

## GaijinPot のビジネスモデル

### 収益構造（BtoB中心）


| 収益源      | 内容                     | 価格帯        |
| -------- | ---------------------- | ---------- |
| 求人掲載料    | 企業が求人を掲載               | ¥15万〜68万/月 |
| 住居掲載料    | 不動産会社が物件掲載（100社以上）     | 非公開        |
| 語学学校広告   | Study セクションでの学校プロモーション | 非公開        |
| Expo 出展料 | 年1回の大型イベント出展（30〜35社）   | 非公開        |
| 記事広告     | ブランド向けタイアップ記事          | 非公開        |


**ユーザー側は基本無料**。求人検索・応募・履歴書作成・住居検索すべて無料で使える。

### ファネル構造

```
Blog記事（SEO集客）
  → サイト訪問（月75万）
    → Jobs / Apartments 利用（無料）
      → 企業が掲載料を払う
```

---

## GaijinPot の主要機能

### Jobs

- 職種 / 地域 / 言語レベル / 勤務形態 / 職務レベルでフィルター
- 企業ページで直接応募、メッセージ機能
- 履歴書作成ツール組み込み
- スカウト機能（企業→求職者へ自動送信、月300〜1,200件）

### Apartments

- リスト / 地図 / ショートリスト（お気に入り）の3表示モード
- フィルター：価格帯、間取り、駅距離、築年数、ペット可、保証人不要等
- 保証人不要・海外からの申込可・テナント保険自動付帯
- 100社以上の仲介業者が物件登録

### GaijinPot Expo（2019年〜）

- 年1回のリアルイベント（東京・秋葉原）
- 2022年：35社出展、500人来場 → 2024年：33社、878人来場
- 「就職フェア」ではなく「コミュニティイベントに企業ブースがある」形
- 無料コンサルティング（ビザ・税金・キャリア・不動産）も併設
- フォトコンテスト等の参加型企画でコミュニティ巻き込み

## 3rd Place が真似すべき5つのポイント

### 1. 「コミュニティ → Expo」モデル

GaijinPotは20年のオンラインコミュニティ運営の後にリアルイベント（Expo）を開始し、出展企業から収益を得た。

**3rd Placeへの応用**:

- すでにカレー会というリアルイベントがある（GaijinPotの逆ルート）
- 将来「3rd Place Expo」的なイベント（メルボルンの日系ビジネス × 渡航予定者）が可能
- GaijinPotの教訓：「就職フェア」でなく「コミュニティイベントに企業ブースがある」形にすると来場者の心理的壁が下がる
- カレー会にパートナー店舗のブースを出すのは自然な延長線上

### 2. 「無料コンテンツで集客 → 有料サービスへ」のファネル

GaijinPotはBlog記事（生活ガイド、ビザ情報、体験談）でSEO集客し、月75万ユーザーを獲得した。

**3rd Placeへの応用**:

- Hubの「知る」タブの業界別ガイドをWebサイト側のブログとしても公開
- 「メルボルン ワーホリ カフェ 仕事」等でSEO流入が見込める
- 無料コンテンツの質が高いほど有料パッケージへの信頼が上がる
- GaijinPotは汎用的。3rd Placeは「メルボルン特化 × 実体験ベース」で差別化

### 3. 「情報の構造化」による資産化

GaijinPotは求人・住居情報を徹底的に構造化し、フィルター検索可能にしている。

**3rd Placeへの応用**:

- 引き継ぎノートをフォームで構造化する方針は正しい（小五郎さんの指摘と一致）
- フィールドの標準化（例：オーナーの人柄→選択式、英語レベル→5段階）
- データが「個人の感想」から「集合知」に変わり、検索性も向上する
- 蓄積されるほど価値が上がるネットワーク効果

### 4. 「お金が見えない」設計思想

GaijinPotはユーザーに無料で使わせ、企業側から取る。ユーザーの目にお金が見えない。

**3rd Placeへの応用**:

- 「コミュニティ内にお金を見せない」という絶対原則と整合する
- 「コミュニティ告知枠 A$50〜200/回」はこのモデルと同じ方向
- Phase 3以降、パートナー店舗30店に増えたら「店舗掲載料」モデルへの移行可能性
- ただし現段階はBtoCメインが正しい（メルボルンは求職者過多で企業に課金動機が薄い）

### 5. 「信頼のブランド化」

GaijinPotが25年かけて構築した外国人コミュニティからの信頼。

**3rd Placeが学ぶべきこと**:

- GaijinPotのフォーラムは匿名口コミが荒れて閉鎖 → 口コミサイト化しない方針は正解
- GaijinPotは大企業買収で「コーポレート感」が出た → 顔が見える小さなコミュニティは差別化要素
- 規模を急がず信頼密度を保つ方が長期的に強い

---

## 真似してはいけないこと


| やるべきでないこと     | 理由                                                 |
| ------------- | -------------------------------------------------- |
| BtoB依存のマネタイズ  | メルボルンでは店舗に課金動機がない（求職者過多）                           |
| 誰でも投稿可能にする    | GaijinPotは規模のために質を犠牲にしている。3rd Placeの信頼は「メンバー限定」にある |
| フォーラム/口コミサイト化 | GaijinPot自身が失敗して2013年に閉鎖した                         |
| 規模で勝負する       | 75万MAU vs 300人は土俵が違う。ニッチ × 深さで勝つ                   |
| 匿名性           | GaijinPotの弱点。3rd Placeは顔が見える関係で克服済み                |


---

## 結論

GaijinPotは「メディア → コミュニティ」の順で25年かけた。3rd Placeは「コミュニティ → メディア/サービス」の順で、すでにコミュニティ基盤がある。これはGaijinPotが一番苦労した部分をスキップできているということ。

あとは **コンテンツの構造化と公開範囲の設計** を磨けば、ニッチだが代替不可能なポジションを確立できる。
