利用規約

バージョン：v2.0 Final（弁護士確認済み）｜最終更新日：2026年4月9日

3rd place hub（以下「当サービス」）をご利用になる前に、本利用規約（以下「本規約」）をお読みください。

第1条（目的・サービス内容）

当サービスは、メルボルン（オーストラリア）での就労・生活を検討する方に対し、以下の情報提供・コミュニティ支援サービスを提供します。

  1. 渡航前ブリーフィング（ビザ・生活情報の案内）
  2. メルボルン在住メンバーとのコミュニティ接続・イベント参加
  3. 渡航先での就労・住居探しに必要な情報・知識・接続機会の提供
  4. 引き継ぎノート（過去の渡航者が作成した就労先情報）の閲覧
重要：当サービスが提供するのは「情報・知識・接続機会」であり、就労先・住居の斡旋・仲介・紹介を行うものではありません。採用・入居の可否は利用者本人と雇用先・家主との間で決定されるものであり、当サービスはその過程に介在しません。

第2条（利用資格）

  1. 日本国籍または日本語を主要言語とする方
  2. メルボルンへの渡航を検討中または渡航予定の方
  3. 本規約に同意された方
  4. 18歳以上の方

第3条（登録と料金）

  1. 利用登録はLINEグループへの参加およびアンケート回答（Stage 0）をもって完了とします。
  2. メルボルン着パッケージ（A$150〜200）の購入は任意です。
  3. 料金は購入時に当サービスが指定する方法でお支払いください。
  4. 一度購入されたパッケージの返金は原則お断りします。ただし、渡航中止の場合は個別に対応します。

第4条（キャッシュバック制度）

  1. 所定の条件（各Stageの完了・アンケート回答・引き継ぎノート作成等）を満たした場合、キャッシュバックをお支払いします。
  2. キャッシュバックの法的性質は「情報提供への謝礼」であり、雇用契約・業務委託契約に基づくものではありません。
  3. キャッシュバックの支払いは、当サービスの判断により変更・終了する場合があります。
  4. キャッシュバックは、メルボルン着パッケージの代金を上限とします（上限：約A$100）。
  5. 引き継ぎノートの作成に伴うキャッシュバックは、ノートの内容・品質を当サービスが確認したうえで支払います。虚偽・不正確な情報が含まれていた場合は支払いの対象外とします。
  6. キャッシュバックの受取人は、受取額が年間の雑所得合計¥20万を超える場合、確定申告が必要となる場合があります。

第5条（引き継ぎノートの取り扱い）

  1. 閲覧目的の限定：引き継ぎノートに含まれる情報は、自身の渡航・就労準備のためにのみ使用できます。
  2. 二次配布の禁止：ノート内の情報を第三者に転送・転載・販売・公開することを禁止します。
  3. 口コミサイト化の禁止：ノート内の店舗情報をSNS・レビューサイト等に掲載・評価・批評することを禁止します。
  4. 店舗名の取り扱い：ノート内に記載された店舗名・オーナー情報は非公開情報として扱い、外部への漏洩を禁止します。
  5. 記載内容のガイドライン：ノートを作成する場合、客観的な事実（時給・勤務時間・業務内容・設備状況等）の記載を推奨します。個人を特定した攻撃・誹謗中傷・差別的表現は禁止します。運営が不適切と判断した記載は削除する場合があります。

上記ルールは、情報提供者・閲覧者・関係者の信頼関係を守るために設けています。違反した場合は利用停止処分とします。

なお、掲載にあたり就労先・家主への事前同意の取得を推奨しますが、必須とはしません。同意を取得せずに掲載した情報に関して生じたトラブルについては、投稿者と当該第三者との間で解決するものとし、当サービスは仲介・調停の義務を負いません。

第6条（禁止事項）

  1. 虚偽の情報を登録すること
  2. 当サービスを通じて知り得た情報を第三者に販売・転用すること
  3. 他の利用者やパートナー店舗への迷惑行為
  4. 法令に違反する行為
  5. 当サービスの運営を妨害する行為
  6. 引き継ぎノートの情報を口コミサイト・SNSに転載すること、または特定の店舗・個人への誹謗中傷を行うこと
  7. 当サービスが「採用・入居を保証した」かのような虚偽の説明を第三者に行うこと

第7条（免責事項）

  1. 当サービスは、採用・入居の成立を保証しません。就労・住居の最終決定は利用者本人と雇用先・家主との間で行われます。
  2. 当サービスは、引き継ぎノートを含む情報の正確性・最新性・完全性を保証しません。
  3. 当サービスは、利用者と雇用先・家主との間のトラブル・損害について責任を負いません。
  4. 当サービスは、法的アドバイスを提供するものではありません。ビザ・税務等は専門家にご相談ください。
  5. キャッシュバック制度は、資金状況等により変更・中止する場合があります。

第8条（個人情報の取り扱い）

個人情報の取り扱いは、別途「個人情報取り扱いに関する同意書」に定めるとおりとします。

第9条（規約の変更）

当サービスは、必要と判断した場合に本規約を変更できるものとします。変更後の規約はLINEグループにて告知し、告知後も利用を継続した場合は変更に同意したものとみなします。

第10条（準拠法・管轄）

本規約は日本法に準拠するものとします。紛争が生じた場合は、東京地方裁判所を第一審の合意管轄裁判所とします。

個人情報取り扱いに関する同意書

バージョン：v2.0 Final（弁護士確認済み）｜最終更新日：2026年4月9日

1. 個人情報の取得項目

  • 氏名・ニックネーム
  • 連絡先（LINEアカウント・メールアドレス等）
  • 渡航予定日・滞在予定期間
  • 希望職種・希望時給・英語力
  • 職歴・スキル等（任意）
  • メルボルン滞在中に取得した就労先情報（引き継ぎノート）※任意

2. 利用目的

  1. メルボルン渡航に関する情報提供・コミュニティ運営
  2. 渡航先での就労・住居探しに必要な情報・知識・接続機会の提供（採用・入居の斡旋・保証は行いません）
  3. キャッシュバックの支払い管理
  4. サービス改善・統計分析（個人を特定しない形）
  5. サービスに関する連絡・案内
  6. 引き継ぎノート作成者へのインセンティブの提供

3. 第三者提供について

（1）接続機会提供のための情報共有

  • 提供先：メルボルン市内の日本食・日本語対応店舗・企業
  • 提供情報：氏名・希望職種・英語力・希望時給・簡単な自己紹介
  • 提供目的：利用者と就労先との接点づくりのため

採否の決定・雇用契約の締結は店舗・企業が独立して行います。当サービスはその過程に介在しません。

（2）その他の提供

法令に基づく場合を除き、上記以外の第三者への提供は行いません。

4. 外国にある第三者への提供

当サービスは、メルボルン（オーストラリア）在住のメンバーおよびパートナー店舗と情報を共有します。

  • 提供先の国：オーストラリア連邦
  • 当該国の個人情報保護制度：Privacy Act 1988が適用されます
  • 提供する情報：氏名・希望職種・英語力・希望時給・簡単な自己紹介

5. 引き継ぎノートの取り扱い

  • ノートの作成は任意です。強制されるものではありません。
  • ノートに含まれる就労先情報は、サービス内の限定メンバーのみが閲覧できる形式で管理します。外部公開・口コミサイトへの転載は禁止です。
  • ノートには客観的な事実の記載を推奨します。特定個人への攻撃・誹謗中傷・差別的表現が含まれる場合、運営の判断で削除します。
  • 店舗名は半匿名形式（カテゴリ・地域・業態のみ表示等）での管理を基本とします。
  • 就労先・家主への事前同意の取得を推奨しますが、必須ではありません。同意未取得で掲載した場合のトラブルは、投稿者と当該第三者間で解決するものとします。

6. 個人情報の管理・保管

  • 情報はSupabase（PostgreSQL）・Notionにて管理します（アクセス権限は運営メンバーのみ）
  • 退会または希望があれば、情報を速やかに削除します

7. 同意の撤回・情報削除の請求

いつでも同意を撤回し、情報の削除を請求できます。

請求先：3rd place hub LINEアカウント（@3rdplacehub）または担当者LINE
