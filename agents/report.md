## 優先改善リスト

### 🔴 優先度：高（すぐやるべき）

#### 1. 次回開催日の掲載（全ペルソナ共通・最重要）

- **問題**：トップページ・Japanページ共に「次のカレー会/イベントはいつか」が一切不明。CTAセクションの「グループ内の告知で次回開催日をご確認ください」は`text-white/25`でほぼ不可視。LINEに入らないと何もわからない構造は、行動直前の離脱を生む
- **改善案**：Hero直下かStats直後に「次回：○月○日（土）@○○」のバナーを設置。手動更新でもよい。または「毎月第○土曜日」などの定例パターンを明示する。Japanページにも同様の次回開催情報を追加
- **関係エージェント**：Agent 1, 2, 3, 5（4体が同一課題を最重要として指摘）

#### 2. LINE URLのプレースホルダー問題（本番前必須）

- **問題**：全ページのCTAボタンが`https://line.me`（LINEトップページ）に向いている。タップしてもコミュニティに参加できず、信頼が一瞬で崩壊する
- **改善案**：実際のLINEグループ招待URLが用意でき次第、全ページで差し替え。プレースホルダーであることはCLAUDE.mdの保留事項に記載済みだが、本番公開前の最優先タスク
- **関係エージェント**：Agent 2, 4, 5

#### 3. ヒーローからLINE直リンクまでの距離短縮

- **問題**：ヒーローの第1CTA「参加する →」は`#cta`アンカーで、ページ最下部まで5セクション分のスクロールが必要。最重要アクション（LINE参加）がファーストビューに存在しない
- **改善案**：ヒーローに「LINEグループに参加する」直リンクボタンを追加、またはFounder/Gallery末尾のインラインCTAをLINE直リンクに変更する
- **関係エージェント**：Agent 2, 5

### 🟡 優先度：中（次のイテレーションで）

#### 4. Hubダミーテスティモニアルの視認性改善

- **問題**：4名分の詳細な口コミ（さくら・けんた・みく・りょうすけ）がリアルに書かれているが実在しない。注釈`text-slate-300 text-xs`は実質的に見えず、本物と誤認されるリスクが高い。誠実さを売りにするコミュニティとして矛盾
- **改善案**：注釈を`text-slate-500 text-sm`に格上げ、またはダミー状態ではテスティモニアルセクションを非表示にする
- **関係エージェント**：Agent 2, 4

#### 5. 既存LINEメンバー向けの導線追加

- **問題**：CTAの主ボタンが「LINEグループに参加する」のみで、すでにメンバーの人には無関係。「次のカレー会の日程を知る」という最も重要なアクションへの導線がない
- **改善案**：CTAセクションに「すでにメンバーの方 → 次回開催日をチェック」のサブリンクを追加。または前述の次回開催日バナーで解決
- **関係エージェント**：Agent 3

#### 6. LINE CTA直下に心理的ハードル軽減のマイクロコピー追加

- **問題**：「LINEグループに参加する」だけでは、入ったら何が起きるかわからない。コミュニティ系LINEへの参加は心理的ハードルが高い
- **改善案**：CTA直下に「グループに入るだけでOK。挨拶不要。次回開催の告知だけ届きます」の一文を添える
- **関係エージェント**：Agent 2

#### 7. スクロールトリガーアニメーションの追加

- **問題**：アニメーションがHeroのページロード時のみ。下スクロールするとページが「静止したまま」で、"かっこいいサイト"との最大の差がここ
- **改善案**：セクションに`opacity-0 translate-y-4`を初期状態とし、Intersection Observerでクラス切替。globals.cssの`animate-fade-up`は既に定義済み
- **関係エージェント**：Agent 6

### 🟢 優先度：低（余裕があれば）

#### 8. Hub `bg-teal-700`/`bg-purple-700` のトークン統一

- **問題**：hub/page.tsxのテスティモニアルに`bg-teal-700`と`bg-purple-700`があり、tailwind.config.tsの定義外の色
- **改善案**：実データ差替え時に`bg-navy`/`bg-orange`等に統一
- **関係エージェント**：Agent 6

#### 9. `text-white/40`のコントラスト改善

- **問題**：japan/hub各ページの説明文に`text-white/40`が多用され、WCAG AAコントラスト比を下回る可能性
- **改善案**：`text-white/60`以上に引き上げ
- **関係エージェント**：Agent 6

#### 10. 3ページ間の統一感強化

- **問題**：Hero高さ（min-h-screen / pt-32 pb-20 / pt-40 pb-28）、背景opacity（0.12/20/10）、Stats列数（4/3/4）が不揃い
- **改善案**：意図的な差別化を維持しつつ、背景opacityとStats列数は統一。フッターのSNSアイコン表示もSubFooterに反映済み
- **関係エージェント**：Agent 6

### Agent 1｜日本在住・渡航検討者の所見

**総評（2〜3文）**
ヒーローの第一文「メルボルンで、仲間と出会い、自分を広げる場所。」はメルボルン在住者向けのメッセージであり、「まだ日本にいてワーホリを迷っている自分」には刺さりにくい。ただしヒーロー直下の「日本からメルボルンを目指している方はこちら」ピルボタンと、Japanページの「渡航前に、先輩の声を聞いておこう。」は非常に的確。全体的にコンテンツの質は高いが、日本在住者が正しいページへ到達するまでに1アクション余分にかかる構造が課題。

**良い点**

- Japanヒーロー「渡航前に、先輩の声を聞いておこう。」は渡航前の不安をそのまま捉えた最良のコピー
- イベント2部構成（帰国者交流 + 英語交流）で「情報収集」と「英語練習」の2大不安を同時解消
- FAQ「まだ決まっていなくても参加できますか？→ 全然アリです」が未決断者の背中を押す
- Stats「¥500 / 毎月 / 東京・喫茶店」が参加コストの低さを3秒で伝達
- Founder「トーストだけ」「具のないラーメン」の具体描写が誕生背景をリアルに

**課題・改善提案**

- ヒーローが日本在住者に向いていない → 副コピーに「まだ日本にいる方へ」を追加、またはピルボタンの視認性強化（`text-white/80`以上）
- 次回イベント日程が不明 → Apply付近に「次回：○月○日（予定）」を追加
- Japanページ内のHub紹介が文脈を断つ → Hub紹介をページ最下部に移動
- LINEとフォームの役割分担が不明確 → 優先度の視覚的差をつける

**このペルソナが離脱しそうなポイント**

- トップページ着地直後（「メルボルンで〜」で始まり「自分はまだ日本にいる」と感じる）
- JapanページのHub紹介セクション（温度差）
- Apply CTAの2ボタン並列（迷い）

**スコア** — 自分ごと化：★★★☆☆、活動の伝達：★★★★☆、行動への誘導：★★★☆☆

### Agent 6｜デザイン品質レビュアーの所見

**総評（2〜3文）**
「かっこよくてプロフェッショナル」という目標に対して現在60〜65点の達成度。Hero（トップ）は洗練されており、navy/orange/creamの骨格は間違いない。しかしセクション間のビジュアルリズムが単調で、タイポグラフィの個性が十分に引き出されておらず、「見た瞬間に息をのむ」体験には届いていない。

**デザインの強み（美的観点）**

- カラーシステムの骨格が正しい（dominant/accent/surfaceの役割分担明確）
- Hero（トップ）の完成度が高い（blur glow + fadeUp + スクロールインジケーター）
- SVGアイコンの一貫使用
- セクションラベルのデザインパターン統一

**デザインの強み（実装品質観点）**

- hex直書きゼロ（全トークン名経由）
- cursor-pointer全クリッカブル要素に付与
- transition-colors duration-200の一貫性
- font-headingのh1/h2/h3適用ルール遵守
- opacity/テキストコントラスト概ね適切

**改善すべき箇所 — 優先度高**

- Hub `bg-teal-700`/`bg-purple-700`がトークン定義外
- `text-white/40`のWCAG AAコントラスト不足（`/60`以上に）
- Statsセクションが3ページとも同一デザイン（ページの個性なし）
- アニメーションがHeroにしか存在しない（最大の課題）
- Navbar以外にインタラクティブな視覚変化がない

**3ページ間の統一感の問題**

- Hero高さ・paddingが不揃い（min-h-screen / pt-32 pb-20 / pt-40 pb-28）
- 背景画像opacityが不揃い（0.12 / 20 / 10）
- Stats列数が異なる（4/3/4）
- 最終CTAのデザイン差（top:bg-navy 2カード / japan,hub:bg-orange センタード）

**「もう一段かっこよくする」ための提案**

1. スクロールトリガーアニメーション（Intersection Observer）— 体験差が最も大きい
2. セクション背景に微細テクスチャ/グレインでLinear・Raycast級の高級感
3. font-headingの文字間隔・行間微調整（日本語タイポグラフィの美しさ向上）
4. Hubのfeatureカードhover強化（`hover:-translate-y-1 hover:border-orange/20`）
5. Japan Hero背景画像をart directionとして活用（clip-path/bg-blend-multiply）

**スコア** — タイポグラフィの個性：★★★☆☆、カラーの効果：★★★★☆、レイアウト・余白：★★★☆☆、実装の精度：★★★★☆、3ページの統一感：★★★☆☆
