# 3rd Place Webサイト 改善レポート（第5回）
**実行日**：2026-04-14
**対象ページ**：/ (トップ)・/japan・/hub
**対象変更**：Founder刷新（教授肩書き・マサカレー→カレー会）・「はじめかた」柔らかトーン化・Gallery/FAQ柔らかトーン化・Final CTA 2カード再構成（写真挿入）

---

## 総評

今回の刷新は全体として確実な前進。特に「宗教っぽさ」の懸念（「みんなのマサさん」「真心」→ 教祖感）は Founder肩書きの「メルボルン在住大学教授」化と「マサカレー→カレー会」の改名によって大きく解消された。柔らかトーン化された「はじめかた」「FAQ」はペルソナ3体すべてから「不安を先回りで潰している」と高評価。Final CTAの2カード構成（在メルボルン vs 渡航検討中）はユーザーが自分の入口に0.5秒で辿り着ける明快な設計として機能している。

一方で、絵文字追加がCLAUDE.mdの「UIアイコンとして絵文字を使わない」ルールに抵触している点はデザインシステムの一貫性を損ねる課題。また Founderセクション内に「マサさん」呼称が残り権威／親しみの人物像がぶれている点、Final CTAカード2の3ボタン構成でLINEボタンが視覚的に埋もれている点は、小さな手直しで大きく改善できる。

## 優先改善リスト Top 5

### 🔴 優先度：高（すぐやるべき）

1. **絵文字をSVGに置き換え（CLAUDE.mdルール違反）**
   - 問題：「はじめかた」の `🌱 🚀`、Final CTAの `☕ ✈️`、Galleryの `🎉 ✨`、FAQの `🙌 🙂 😊` などがCLAUDE.mdの「SVGアイコンのみ・絵文字をUIアイコンとして使わない」ルールに抵触
   - 改善案：装飾として入れた絵文字を既存SVG（芽/ロケット/カップ/飛行機/カレンダー等）に差し替え。国旗絵文字（🇦🇺🇯🇵）は許容
   - 関係エージェント：Agent 6

2. **Final CTAカード2のLINEボタンを視覚的に格上げ**
   - 問題：「これからメルボルンに行く方」カードで最重要アクションである「LINEグループ参加」が`underline`のサブリンクとして最下部に埋もれている。迷ったユーザーの最低抵抗ルートなのに視認性が最も低い
   - 改善案：小さめのoutlineボタン（`bg-white/15 border border-white/30 text-white px-4 py-2 rounded-xl`）に変更
   - 関係エージェント：Agent 1, Agent 5

3. **Founder Origin Story内の「マサさん」→「野口さん」に統一**
   - 問題：Founderの肩書きを「野口マサ / 大学教授」で権威づけた直後、本文で「マサさんの親心から」と愛称に戻り、権威↔親しみの人物像が一貫性を欠く
   - 改善案：Origin Story本文の「マサさん」1箇所を「野口さん」に変更
   - 関係エージェント：Agent 4

### 🟡 優先度：中（次のイテレーションで）

4. **Heroの「日本からメルボルンを目指している方はこちら」ピルリンクを格上げ**
   - 問題：ファーストビューで日本在住ペルソナを掴む最重要導線なのに、サブリンクとして埋もれ、モバイル流し読みで見落とされる
   - 改善案：オレンジ系装飾（`border-orange/60 text-orange`）に変更、or 文言を「渡航準備中の方はこちら（東京イベント・渡航サポート）→」に変更
   - 関係エージェント：Agent 1, Agent 5

5. **既存メンバー向けFAQを1問追加**
   - 問題：FAQ全問が初参加者向けで、LINEに入っているがブランクがある既存メンバーが「自分のこと」を見つけられない
   - 改善案：「もう数ヶ月経つけど、今から行っても大丈夫？」→「全然OK！毎回初めましての方が5〜10人いるし、途中から来た人もたくさんいますよ」を追加
   - 関係エージェント：Agent 3

### 🟢 優先度：低（余裕があれば）

- Founderセクションの`text-white/50`段落のコントラスト底上げ（Agent 6、WCAG AA相当）
- Galleryカードdescに雰囲気の一言追加（鍋会「毎年恒例、気づいたら3時間経ってた」等／Agent 3）
- Final CTAカード1に次回開催日を直接表示（#next-eventへのスクロール不要化／Agent 2）
- Hub テスティモニアルのダミー注記を`bg-orange/10`バナーに格上げ（Agent 4）
- Founderの「真心・世界を変える」段落の温度感調整（Agent 2懸念。Agent 4は問題視せず。判断保留）
