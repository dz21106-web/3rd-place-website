## 優先改善リスト

### 🔴 優先度：高（すぐやるべき）

#### 1. Japanページ申し込みフォームURLの修正または非表示化
- **問題**：`href="https://docs.google.com/forms"` がプレースホルダーのまま。CTAクリック後にGoogle Formsトップに飛び、信頼が崩壊する
- **改善案**：正式URLが未整備なら、フォームボタンを削除しLINE誘導に一本化。または「準備中」ラベルに差し替え
- **関係エージェント**：Agent 1, 4, 5（3エージェントが指摘）

#### 2. トップページからJapanページへの導線強化
- **問題**：日本在住ペルソナへの入口がHero下のピルリンク1本のみ。スマホで気づかず「メルボルン在住者向けサイトだ」と離脱するリスクが高い
- **改善案**：Statsセクション直後または「はじめかた」Step 2の東京イベント箇所にJapanページへの誘導カードを追加
- **関係エージェント**：Agent 1, 5

#### 3. 3ページ間のHero演出格差の解消
- **問題**：トップのみグロー効果・スクロールインジケーター・animate-fade-up連鎖があり、Japan/Hubは演出が薄い。「別サイト感」が出ている
- **改善案**：Japan/HubのHeroにグロー効果追加、CTAに `hover:-translate-y-0.5 hover:shadow-lg` 追加、最低限のアニメーション連鎖を追加
- **関係エージェント**：Agent 6

### 🟡 優先度：中（次のイテレーションで）

#### 4. Hubダミーテスティモニアルの表示改善
- **問題**：詳細な体験談が書かれているが末尾に小さく「ダミーです」。読み進めてから「嘘だった」と気づく構造は信頼破壊
- **改善案**：カードデザイン自体にグレーアウト/ロック処理を施すか、テスティモニアルセクションを一旦削除
- **関係エージェント**：Agent 2, 4

#### 5. デザイントークン違反の修正
- **問題**：
  - Hub テスティモニアルに `bg-teal-700`, `bg-purple-700`（トークン外の色）
  - Japan に `📍` 絵文字（SVGアイコンルール違反）
  - `text-white/55`（非標準opacity、/50か/60が正規）
- **改善案**：teal/purple → navy/orange系に変更、📍 → SVGアイコン、/55 → /50に修正
- **関係エージェント**：Agent 6

#### 6. 既存メンバー向けの「一人でも大丈夫」メッセージ強化
- **問題**：安心メッセージがHow to Joinのステップ2に埋もれている。ギャラリーにも初参加者の雰囲気を示すキャプションがない。FAQに「一人で行っても大丈夫？」がない
- **改善案**：
  - ギャラリー写真のキャプションに「この日初参加」「LINEで初めて会った」等を追加
  - FAQに「初参加で知り合いゼロでも大丈夫？」を追加
  - ギャラリー直前にも「一人で来る方がほとんど」を繰り返す
- **関係エージェント**：Agent 3

#### 7. 参加者の「生の声」の追加
- **問題**：写真は豊富だが、参加者の声・顔出しコメントが一切ない。「誰が・どんな経験で・何を話しているか」がわからない
- **改善案**：帰国者・参加者の1〜2行テキスト引用（顔写真+名前+一言）を1〜3件追加
- **関係エージェント**：Agent 1, 2, 3

### 🟢 優先度：低（余裕があれば）

#### 8. Hub→Japan間の相互リンク追加
- **問題**：HubからJapanへの直接リンクがない。日本からの渡航検討者がHubを見た後、Japanイベントへ誘導する経路がない
- **改善案**：Hub Final CTAの2ボタン目を「東京イベントはこちら」に変更、またはSubFooterで補完
- **関係エージェント**：Agent 5

#### 9. メルボルン在住者への独自価値訴求
- **問題**：サイト全体が「渡航前〜到着直後」向けコピーが主体。「先輩側として貢献できる」ポジティブアピールが弱い
- **改善案**：「すでに在住の方は『先輩』として後輩の力になれます」等の一文を追加
- **関係エージェント**：Agent 2

#### 10. セクション間のソフトセパレーター
- **問題**：bg-cream → bg-white → bg-navyの切り替わりがハード
- **改善案**：各セクション末尾に `h-px bg-gradient-to-r from-transparent via-gray-200 to-transparent` のセパレーター追加
- **関係エージェント**：Agent 6

## エージェント別詳細所見

### Agent 1｜日本在住・渡航検討者の所見

**総評（2〜3文）**
トップページのヒーローコピー「メルボルンで、仲間と出会い、自分を広げる場所。」は美しいが、日本にいる渡航検討者にとって「自分ごと」としてピンとくるには一歩足りない。一方、Japanページは「渡航前に、先輩の声を聞いておこう。」という直球コピーが刺さる上に、2部構成の具体性・¥1,000という低価格・英語不安へのフォローが揃っており訴求力は高い。ただし、トップページからJapanページへの導線が一本の細いピルリンクのみで、離脱リスクが大きい。

**良い点**
- Japanページヒーロー「渡航前に、先輩の声を聞いておこう。」が不安を持つペルソナに直接刺さる
- オレンジバナーで日付・時間・会場が一目でわかる具体感
- Part 2の英語不安解消コピー「英語が得意でなくてもOK」
- FAQ「ワーホリの予定がまだ決まっていなくても参加できますか？」
- 3ステップフロー「所要時間：10秒」「挨拶不要」のマイクロコピー

**課題・改善提案**
- トップページヒーローが「すでにメルボルンにいる人」向けの印象 → 「渡航前から繋がれる」の一言追加
- トップ→Japanへの導線がピルリンク1本のみ → Stats直後にJapan誘導カード追加
- ¥1,000の価値訴求がない → 「カフェ1杯分で渡航後の不安が変わる」のようなコピー追加
- 申し込みフォームURLがプレースホルダー → LINE一本化かURL修正

**スコア** 自分ごと化：★★★☆☆ / 活動の伝達：★★★★☆ / 行動への誘導：★★★☆☆

### Agent 2｜メルボルン在住・未参加者の所見

**総評（2〜3文）**
「誰でも歓迎」メッセージと低ハードルの参加設計は高評価。ただし「すでにメルボルンにいる自分が参加する理由」が明示されておらず、「渡航前の人向けサービス」の印象が勝る。Hubページは「開発中」+「ダミーテスティモニアル」で「今すぐ使えるものがない」と感じさせるリスクがある。

**良い点**
- 「一人で来る方がほとんど」の明記が内輪感への不安を解消
- 3ステップの参加フローがシンプルで「今週末試せる」感がある
- 「入るだけでOK。挨拶不要」のコピーが警戒心を和らげる
- マサさんのカレー誕生秘話が感情に刺さる

**課題・改善提案**
- メルボルン在住者への独自メリット訴求が薄い → 「先輩として貢献できる場」アピール追加
- 参加者の声・具体エピソードが不足 → テキスト引用1〜2件追加
- チームの「投資家対応」「KPI管理」がビジネス臭 → より柔らかい表現に

**スコア** 参加しやすさ：★★★★☆ / 活動の伝達：★★★☆☆ / 行動への誘導：★★★★☆

### Agent 3｜既存メンバー・未参加者の所見

**総評（2〜3文）**
「まず来てみてください」という背中を押す設計をしっかり意識している。Stats実績数字は安心感に直結。ただし「すでにLINEメンバーだけど行けていない」という状況をピンポイントに射抜くコンテンツが少なく、先送りしやすい。

**良い点**
- 次回開催日がオレンジバナーで即座にわかる
- ギャラリー30枚・カテゴリタブで多彩なイベントの楽しさが伝わる
- Final CTAの「すでにメンバーの方 → 次回開催日をチェック」が既存メンバーへの配慮

**課題・改善提案**
- 「LINEにいるけどまだ来たことない方へ」の語りかけがない → バナー近くに追加
- 安心メッセージの位置が深い → ギャラリー直前にも繰り返す
- ギャラリーに「初参加者」の雰囲気を示すキャプションがない
- FAQに「一人で行っても大丈夫？」がない

**スコア** 参加意欲の喚起：★★★★☆ / 安心感の醸成：★★★☆☆ / 行動への誘導：★★★☆☆

### Agent 4｜信頼構築チェッカーの所見

**総評（2〜3文）**
実名・写真・具体的な数字を惜しみなく開示しており、コミュニティの実在感は十分。マサさんのストーリーは★5の説得力。ただしHubダミーテスティモニアル・フォームURLプレースホルダーが信頼の積み上げを途中で止めている。

**信頼を高めている要素**
- 具体的数字（300人・20回以上・毎月30〜40人）
- 創設者実名・写真・感情的ストーリー（★5）
- 運営チーム4名の実名・顔写真・役割
- 「仕事保証はしない」免責宣言の誠実さ
- 「開発中」バッジの正直な表示
- SNS実リンク（Instagram・Facebook）

**信頼を損なっているリスク**
- ダミーテスティモニアルが目立つ位置に掲載（「嘘だった」構造）
- 申し込みフォームURLがプレースホルダー
- イベント日時のハードコード（過去日になると放置サイトに見える）
- 「公認推薦状」の権威根拠が不明

**スコア** 社会的証明：★★★★☆ / 透明性：★★★★☆ / ストーリー説得力：★★★★★ / Hub誠実さ：★★★☆☆

### Agent 5｜行動導線チェッカーの所見

**総評（2〜3文）**
LINE参加への導線は一貫して設置されており、Hero・バナー・Final CTAの複数箇所に実URLが貼られている点は高評価。Japanの申し込みフォームプレースホルダーが最終CVに穴を開けており、Hub→Japan間の回遊性が手薄。

**うまく機能している導線**
- Hero 2択ピルボタン（LINE参加=orange, 3rd Placeとは=白）
- Next Eventオレンジバナーの即時行動設計
- Final CTA 2択カード（メルボルン/日本）
- About末尾のHub誘導ブロック
- Navbarの常時表示「参加する」CTA

**導線の問題・改善提案**
- Japan申し込みフォームがデッドリンク（重大）
- Hubページ中間にインラインCTAがない
- Hub→Japan間の直接リンクがない

**スコア** CTAの明確さ：★★★★☆ / ページ間の回遊性：★★★☆☆ / 摩擦の少なさ：★★★★☆

### Agent 6｜デザイン品質レビュアーの所見

**デザインの強み**
- Heroのグロー演出（bg-orange/10 blur-[120px]）
- タイポグラフィの3層ヒエラルキー一貫性
- GallerySectionのホバーキャプション + マソンリー風変化
- SVGアイコン完全統一・cursor-pointer徹底・transition一貫性

**改善すべき箇所**
- 📍絵文字（Japanバナー）→ SVG化
- Hub テスティモニアルに `bg-teal-700`, `bg-purple-700`（トークン外）→ navy/orange系に
- Hub「開発中」バッジ2つ重複 → 1つに統合
- `text-white/55` → `/50`に修正

**3ページ間の統一感**
- トップのみグロー・アニメーション連鎖 → Japan/Hubにも追加
- CTAのhoverエフェクト → Japan/Hubにも `hover:-translate-y-0.5 hover:shadow-lg` 追加
